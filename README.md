# Comic-Con-Parking-System

This project contains an ER diagram for a multi-zone parking system used during a large event like Comic-Con.

## ER Diagram Image:
<img width="1600" height="1048" alt="Comic-Con-Parking-System_dark" src="https://github.com/user-attachments/assets/dbe94003-c700-420e-a53a-86abd1f3ff64" />

It supports:
	•	Managing different types of vehicles (bike, car, SUV, EV)
	•	Organizing parking spots across zones and levels
	•	Handling reserved categories like VIP, staff, exhibitors, EV charging
	•	Tracking parking sessions with entry and exit time
	•	Issuing tickets for each parking session
	•	Recording payments for parking

Key points:
	•	A vehicle can have multiple parking sessions
	•	A parking spot can be reused across different sessions
	•	Each parking session has one ticket and one payment
	•	Parking spots are organized by zones and categories

The ER diagram image is included in this repository.
